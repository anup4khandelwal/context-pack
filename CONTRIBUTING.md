# Contributing to context-pack

Thanks for your interest in contributing!

## Quickstart

```bash
npm install
npm test
npm run build
```

## Development workflow

- Make changes on a branch
- Keep changes small and focused
- Add/adjust tests when behavior changes
- Run `npm test` and `npm run build` before opening a PR

## Commit and PR guidelines

- Prefer descriptive commit messages
- Link issues when relevant
- Describe what changed and why in the PR

## Project structure

- `src/` core CLI
- `rules/` heuristics configuration
- `integrations/` MCP/GitHub Action/VS Code wrappers
- `docs-site/` VitePress docs
